# Project Recursion

This project is part of The Odin Project JavaScript exercise.

The goal of this project is to practice recursion by building two small programs:

1. Fibonacci sequence
2. Merge sort

## Fibonacci

The `fibonacci.js` file contains two functions:

- `fibs(n)` creates a Fibonacci sequence using iteration.
- `fibsRec(n)` creates a Fibonacci sequence using recursion.

## Merge Sort

The `mergeSort.js` file contains a recursive merge sort function.

Merge sort works by splitting an array into smaller arrays, sorting those smaller arrays, and then merging them back together in order.
